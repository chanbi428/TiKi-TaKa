Tiki-Taka(Voice calendar)
================
### 음성인식 API를 활용한 일정관리 반응형 웹 서비스
#### ( 4인 팀프로젝트 )
___
+ ### 개발환경
  + BE
    + 서버:Express
    + DB:MySql(Sequalize)
  + FE
    + Next.js
    + React.js
    + UI Framework:Ant-Design
  + 역할
    + Media Query와 Ant-Design을 활용한 반응형 웹 구현
    + React Framework를 통한 컴포넌트 구현
___
